# How to clone a repository to your local machine? #
*[Home](README.md)*  

**Clone** is used to create a repository on your local machine for the first time:

* Visit and log into [GitHub](https://github.com/Digiata/TrainingStadium8Backend)
* Navigate to the repository that you want to clone, click the Code dropdown button on the top right and copy the HTTPS link
	+ [Stadium8TrainingBackend](https://github.com/Digiata/TrainingStadium8Backend)
* Open a new tab at the top of SourceTree using the + icon
* Click the Clone Button at the top left of the screen
* Paste the link you copied from GitHub and paste it into the Source Path
* Choose an appropriate destination path as shown in the example below
	
	* **Standard Local Directory Path:**
	
		* *C:\DigiataRepos\\"GitHub Project Name"\\"GitHub Repo Name"*
	
* Click the Clone button
* The repository should now be successfully cloned onto your local mahcine in the target directory that you specified
